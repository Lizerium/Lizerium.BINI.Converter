Freelancer BINI guide

How to open and convert BINI files

How to open BINI files in Freelancer

BINI files are binary configuration files used by Freelancer. Convert them to INI text before editing ship, market, equipment, or universe data.

BINI to INI converter

Drag a file into the converter, keep Auto mode, and download readable text output. The conversion runs inside your browser.

INI to BINI packing

After editing INI text, choose INI text to BINI and save a binary file that can be used by existing Freelancer tooling.

Troubleshooting

Common errors when opening BINI files

File is not readable

Check that the file is a Freelancer BINI or plain INI file and that it was not truncated during download or extraction.

Encoding issues

Most Russian Freelancer files need Windows-1251. Try Latin-1 or UTF-8 only when the text looks broken.

Corrupted data

Invalid BINI headers, missing string-table data, or unsupported values can prevent conversion.
